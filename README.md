# Great Learning - Udemy Clone

Great Learning is a responsive online learning platform inspired by Udemy, built using HTML and CSS. The project showcases a modern educational website interface with course recommendations, popular courses, category navigation, and a responsive layout.

## Features

* Responsive Navigation Bar
* Course Search Interface
* Category Menu
* Recommended Courses Section
* Popular Courses Section
* Topic Recommendations
* Modern UI Design
* Responsive Layout for Multiple Screen Sizes
* Structured Footer Section

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Font Awesome

## Project Goal

The goal of this project is to strengthen front-end development skills by recreating a professional e-learning platform interface while practicing responsive web design and modern layout techniques.
